## eslintrc  

El archivo `.eslintrc.json` contiene configuraciones para **ESLint**, una herramienta que ayuda a mantener un código limpio y sin errores en proyectos JavaScript.  

### Configuración

```json
{
  "env": {
    "browser": true,
    "es2021": true
  },
  "extends": "eslint:recommended",
  "parserOptions": {
    "ecmaVersion": 12,
    "sourceType": "module"
  },
  "rules": {
    "semi": ["error", "always"],
    "quotes": ["error", "double"]
  }
}
